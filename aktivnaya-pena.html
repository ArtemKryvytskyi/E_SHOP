4. Как применять бесконтактный автошампунь для мойки авто в домашних условиях?
На самом деле качественно помыть автомобиль можно (и нужно) без особых усилий в домашних условиях, не используя аппарат высокого давления с пенным комплектом.
В домашнем арсенале нужно иметь:
- распылитель, через который наносится приготовленный бесконтактный автошампунь
- сам концентрат для бесконтактной мойки авто, активная пена автошампунь для ручной, основной мойки (обязательно)
Хотелось бы наперёд оговориться - дешёвые распылители работать нормально не будут. Лучше потратиться и купить один раз профессиональный инструмент. Мы используем марки Epoca. Есть другие марки распылителей. На момент покупки мы других не нашли.
Помните, шампунь для бесконтактной мойки – это очень агрессивный щелочной состав.
Рецепт (правильное нанесение активной пены):
Берем 2 литра воды, добавляем 2% концентрата активной пены, в данном случаи мы применяем бесконтактный шампунь Cargo 4100 Forte (2% = 40 мл). Смешиваем, через распылитель наносим готовое средство на кузов автомобиля. Начинать наносить раствор необходимо с нижней части авто и постепенно переходить выше (снизу вверх). 1 - 2 минуты ждём пока активная пена делает свою работу, смываем.

Шампунь для бесконтактной мойки.

Cargo 2500 - это бесконтактный автошампунь для предварительной мойки авто. Самый низкоконцентрированный шампунь для бесконтактной мойки в линейке Kenotek. Слабая концентрация этого состава имеет определенное значение и смысл. Почему и зачем мы опишем ниже.
Активная пена Cargo 2500 представляет собой концентрированное щелочное пенное чистящее средство для бесконтактной мойки наружных поверхностей. Это профессиональный бесконтактный автошампунь для автомойки. Лучшее предложение для автомоек самообслуживания. Этот шампунь для бесконтактной мойки применяется для всех типов транспортных средств.
Cargo 2500 содержит ингредиенты, признанные в пищевой промышленности в процессах очистки (по методологии НАССР) (чистка рефрижераторов, холодильников)
Дозировка: 2 - 7 %
pH - 11.5 (1%)
Автошампунь абсолютно безопасен для автомобиля и никак не вредит лакокрасочному покрытию.
Шампунь для бесконтактной мойки Cargo 2500 - это самый распространенный автошампунь для автомоек самообслуживания. Средство менее концентрированное, чем его аналоги в линейке Kenotek. Но концентрация полностью компенсируется стоимостью самого средства.
Что это означает?
Например

шампунь для бесконтактной мойки Cargo 4100 Forte,

имеет концентрацию 0,5-2%. Соответственно, для обработки одного автомобиля необходимо затратить 35 мл концентрата. Cargo 2500 имеет концентрацию 2-7%, соответственно порция концентрата этого автошампуня увеличится в 3-ри раза по сравнении с Cargo 4100 Forte, будет составлять примерно 100-120 мл.
Почему это необходимо и очень важно? Многие автоматические станции самообслуживания не могут (по своим техническим характеристикам) определить такую маленькую дозу активной пены (35 мл концентрата), и выдают шампунь в большем количестве. Это в свою очередь приводит к ненужному перерасходу концентрата шампуня для бесконтактной мойки.
Бесконтактный автошампунь Cargo 2500 из-за своей концентрации (2-7%) предотвращает случайный перерасход средства и прощает погрешности при дозировке, которые может допустить как человек так и автоматика.
Если наносить шампунь для бесконтактной мойки Cargo 2500 на автомобиль через ( или с помощью) пенного пистолета используя аппарат высокого давления, получается следующая дозировка:
размешиваем концентрат автошампуня для предварительной чистки поверхности в пропорции 1:5, 1:4 или 200-300 мл Cargo 2500 + 700-800 мл воды = 1 литр готового раствора автошампуня;
Полученный бесконтактный автошампунь наносим на автомобиль. Наносить необходимо начиная с нижней части авто и постепенно переходить выше. Если соблюдать технологию и правильно наносить шампунь для бесконтактной мойки, из 1-го литра готового раствора можно помыть минимум 3-ри автомобиля бизнес класса.
В результате получаем: для обработки одного автомобиля необходимо до 100 мл концентрата активной пены Cargo 2500.
Стоимость одной автомойки таким средством не превышает 4 грн.
Хотим обратить Ваше внимание, что любая активная пена для автомобиля, в независимости от производителя - это средство для предварительной мойки автомобилей. Это бесконтактная обработка кузова автомобиля, цель которой удалить крупные загрязнения и подготовить автомобиля для нанесения основного моющего средства, автошампунь для ручной мойки.

Активная пена

- это состав или шампунь для предварительной обработки. Это не средство 3 в 1: предварительная мойка + шампунь + воск.
